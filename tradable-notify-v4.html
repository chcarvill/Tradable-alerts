Step 1 — API configuration
TwelveData: free tier gives 800 API calls/day — plenty for polling every 5 minutes. Register at twelvedata.com and copy your API key above.

EmailJS setup (free, 200 emails/month):
1. Go to emailjs.com → Sign up free
2. Add an Email Service (Gmail works great) → copy the Service ID
3. Create an Email Template — include these variables: {{subject}}, {{message}}, {{to_email}}
4. Copy the Template ID
5. Go to Account → Public Key and copy it
Paste all three above, enter your email address, then hit ✉ Test Email.

Volume note: TwelveData returns tick volume (number of price changes per candle) for Forex — not true traded volume. It's a reliable proxy for momentum and conviction, used widely across retail platforms.
